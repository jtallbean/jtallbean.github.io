What is Web Development?

Web development, colloquially called web dev, is a term used to describe the work needed to create a website. It encompasses a wide variety of tasks and designs, and can range from simple single-page sites to complex applications and networks. Web dev entails the creation of content management systems, which act as a buffer between the database itself and the user in the browser.

What can you do?

Websites can achieve many different purposes and serve different functions. Examples include online marketplaces such as Amazon, or forums such as stackoverflow. But each of these webpages rely on the same basic principles.

How can you do it?

Modern webpages are created using HTML, or hypertext markup language. HTML defines the structure of the page. Cascading style sheets, or css, defines the appearance of the page. However, in order to create responsive webpages, a language such as Javascript is needed. Javascript controls the behavior of a page.
