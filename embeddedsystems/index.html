Embedded Systems Programming

Used for home automation and controlling sensors.

Arduino

I used Arduino to build a wheeled robot to self identify targets along a dynamic path. Programmed in Arduino and focused on PID control to drive and properly position the robot, as well as interface with various light sensors. Information is transmitted back to a computer using an onboard RF transmitter.

Raspberry Pi

I used a Raspberry Pi in conjunction with a Kinect 360 sensor to automate garage door operation. Programmed in Python and used the depth sensor on the Kinect to determine distance from an approaching vehicle. A set of addressable RGB lights connected to the Raspberry Pi are used to indicate distance to the driver for precise parking.
