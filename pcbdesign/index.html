PCB Design

The part that allows everything else to actually work

Circuit Design

Circuits are designed and prepared for production using KiCad. In order to design a circuit each part is taken into account, which involves reading datasheets and other specifications for routing. As an example care must be taken for USB and other high speed bus routing, as well as capacitor placement for proper bypassing.

Validation/Testing

Each PCB is ultimately tested and validated for errors in production and design.

Firmware

In order to use the PCBs each controller must be flashed with firmware. For example a commonly used Atmega32u4 controller in a keyboard PCB. Writing the firmware involves C programming, explicitly dealing with low level memory addresses and space used.
